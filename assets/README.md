Project assets

This folder holds project assets used by the frontend and for project delivery.

Structure

- 3d_parts/   - Store 3D design files STL format. Keep original source files here.
- reports/    - Store project reports, PDFs, and documentation artifacts.
- images/     - Store images used in the project and documentation.

Guidelines

- Prefer descriptive filenames: e.g. `clamp.stl`, `Rubik_s_Cube.pdf`, `block_diagram.png`.
